### git上传代码到github

#### 进入项目目录，右键“Git Bash hear”
git init

#### 将目录下所有文件上传，“.”代表所有文件，可替换成具体文件名
git add .

#### 将项目提交到github
git commit -m "本次提交注释"

#### 将代码关联到github
git remote add origin https://github.com/qw52655265/my-react.git

#### 上传代码到github先pull（更新）
git pull origin master

#### 上传代码到远程仓库
git push -u origin master
